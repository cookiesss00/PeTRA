# PeTRA: Period Tracking & Health Analysis

PeTRA is an Android-based menstrual health tracking application designed to provide users with accurate cycle predictions, symptom logging, and personalized health insights[cite: 1]. Built with a focus on user privacy and intuitive design, PeTRA empowers individuals to take control of their reproductive health[cite: 1].

## 🚀 Key Features

*   **Intelligent Calendar:** Visualize menstrual cycles, ovulation windows, and historical symptom data through a dynamic calendar interface[cite: 1].
*   **Smart Reminders:** Automated notification scheduling to alert users of upcoming cycles, ensuring proactive health management[cite: 1].
*   **Personalized Mood Booster:** A logic-based recommendation engine that suggests wellness tips based on daily mood and flow intensity logs[cite: 1].
*   **Diary Integration:** A dedicated space for daily reflections, lifestyle factors, and symptom tracking[cite: 1].
